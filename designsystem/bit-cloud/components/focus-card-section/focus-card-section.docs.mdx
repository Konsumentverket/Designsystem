## Introduction

The focus card component is a reusable UI element designed for React applications, providing users with interactive clickable elements.

import { FocusCardSection } from './focus-card-section';

### Instructions
#### Installation

```js
npm i @konsumentverket-sverige/designsystem.focus-card-section@0.0.1
```

#### Importing

```js
import FocusCardSection from '@konsumentverket-sverige/designsystem.focus-card-section@0.0.1';
```

### Component identifier
```js
data-comp="focus-card-section"
```


### Usage:

#### As focus card
```js
<FocusCardSection />
```
