## Basic usage

Use the link card component:

```js
<LinkCard />
```

### Display the text

```js live
<LinkCard
    id="one-docs" 
    href="#" 
    text="Lorem ipsum">
    <p>
        Lorem ipsum dolor sit amet
    </p>
</LinkCard>
```
