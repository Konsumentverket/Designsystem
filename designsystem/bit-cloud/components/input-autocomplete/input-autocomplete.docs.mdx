## Introduction

The InputAutocomplete component is a reusable UI element designed for React applications, providing users with interactive clickable elements. 

import { InputAutocomplete } from './input-autocomplete';

### Instructions
#### Installation

```js
npm i @konsumentverket-sverige/designsystem.input-autocomplete@0.0.1
```

#### Importing

```js
import InputAutocomplete from '@konsumentverket-sverige/designsystem.input-autocomplete@0.0.1';
```

### Component identifier
```js
data-comp="input-autocomplete"
```


### Usage:

#### As InputAutocomplete
```js
<InputAutocomplete {...props}/>
```
