## Basic usage

Use the link text card component:

```js

```

### Display the text

```js live

    id="one-docs-two"
    href="#"
    text="Lorem ipsum">
    <p>
    Lorem ipsum dolor sit amet
    </p>
</LinkTextCard>
```
