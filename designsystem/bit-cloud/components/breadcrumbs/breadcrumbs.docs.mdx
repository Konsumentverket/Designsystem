## Introduction

The breadcrumbs component is a reusable UI element designed for React applications, providing users with interactive clickable elements. It can serve as both a standard button and a link. The appearance of the button can be easily customized using Emotion styling. Additionally, it supports onClick event handling to trigger actions.

import { Breadcrumbs } from './breadcrumbs';

### Instructions
#### Installation

```js
npm i @konsumentverket-sverige/designsystem.breadcrumbs@0.0.1
```

#### Importing

```js
import Breadcrumbs from '@konsumentverket-sverige/designsystem.breadcrumbs@0.0.1';
```

### Usage:

#### As breadcrumbs
```js
<Breadcrumbs items={breadcrumbItems} />
```
