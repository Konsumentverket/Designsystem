import { SubHeading } from './sub-heading';

## Basic usage

Use the sub heading component:

```js live
<SubHeading level={1} text="h1 Lorem ipsum" headingLine richText={false} />
```

```js live
<SubHeading level={3} text="h3 Lorem ipsum" richText={false} />
```

```js live
<SubHeading level={1} text="h1 with styleLevel 4" styleLevel={4} richText={false} />
```

```js live
<SubHeading level={2} text="Richtext h2" richText={true} />
```
