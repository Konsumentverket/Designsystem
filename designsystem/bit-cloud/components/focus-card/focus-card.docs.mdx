## Introduction

The focus card component is a reusable UI element designed for React applications, providing users with interactive clickable elements.

import { FocusCard } from './focus-card';

### Instructions
#### Installation

```js
npm i @konsumentverket-sverige/designsystem.focus-card@0.0.1
```

#### Importing

```js
import FocusCard from '@konsumentverket-sverige/designsystem.focus-card@0.0.1';
```

### Component identifier
```js
data-comp="focus-card"
```


### Usage:

#### As focus card
```js
<FocusCard />
```
